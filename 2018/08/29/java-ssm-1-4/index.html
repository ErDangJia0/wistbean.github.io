Github

全球最大的社交编程网站！

Github 有什么用？

  1. 代码托管
  2. 多人协作
  3. 个人博客
  4. 个人简历

使用Github

  • star 收藏
  • fork “复制”
  • repository 仓库
  • watch 观察
  • Gist 代码片段
  • Pull Request 请求合并
  • Issue 提问题/bug
  • clone 克隆
  • git config 配置信息
  • git push 将本地代码推送到github上去
  • git pull 将远程的代码更新下来
  • git branch 分支

  • git merge 合并

    ssh授权

    生成秘钥：ssh-keygen -t rsa
    验证是否授权成功： ssh -T git@github.com
